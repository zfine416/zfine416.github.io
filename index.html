StockTweets

StockTweets allows a user to search a stock based on its ticker symbol and then returns a word cloud based on the most trending words related to that stock. Additionally StockTweets provides users with current information related to that stock.

Technologies Used: Ruby on Rails, Javascript/jQuery, Yahoo Finance Gem, Twitter API, HTML5/CSS3, Sass and Authentication, Bootstrap

  • StockTweets Heroku Website
